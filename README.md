# Personal Portfolio — Maheshkumar Selvakumar

My personal portfolio website showcasing my projects,
skills and journey as a Full Stack Developer.

## Live Site
[https://maheshkumarselvakumar.github.io/portfolio/](https://maheshkumarselvakumar.github.io/portfolio/)

## Features
- Sakura blossom theme with CSS variables
- Live GitHub API integration — fetches real profile stats
- Smooth scroll navigation
- CSS Grid and Flexbox layouts
- Font Awesome brand icons with official brand colours
- Mobile first responsive design
- Hover animations and transitions
- JavaScript DOM manipulation
- Time-based greeting using JavaScript

## Tech Stack
- HTML5 — semantic structure
- CSS3 — custom properties, Flexbox, Grid, animations
- JavaScript — DOM manipulation, Fetch API, localStorage
- GitHub REST API — live profile data
- Font Awesome — skill and contact icons
- Google Fonts — IBM Plex Serif
- Deployed on GitHub Pages

## Sections
- About Me — profile photo, bio, location badges
- Goals — three card grid with hover effects
- Skills — brand icon cards with official colours
- Projects — NEXUS live demo and code links
- GitHub Stats — live API data
- Contact — email and GitHub links

## Local Setup
```bash
git clone https://github.com/MaheshkumarSelvakumar/portfolio.git
cd portfolio
```

Open `index.html` with Live Server in VS Code.
